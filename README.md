# 💄 La Cosmetica - Application Web (Frontend + API)

## 📌 Description

**La Cosmetica** est une application web dédiée à la gestion et à la consultation de produits pharmaceutiques et cosmétiques.

Ce projet est composé de deux parties :
- 🔙 **Backend API** développé avec Laravel
- 🎨 **Frontend** développé avec React JS

L’objectif est de permettre aux utilisateurs de :
- Consulter les produits disponibles
- Passer des commandes
- Gérer leurs commandes
- Accéder à une interface moderne et intuitive

---

## 🛠️ Technologies utilisées

### 🔙 Backend (API)
- PHP / Laravel
- Laravel JWT (authentification)
- PostgreSQL 🐘

### 🎨 Frontend
- React JS (Vite)
- Axios (communication API)
- Tailwind CSS (design)
- React Router DOM (navigation)
